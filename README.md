#### Developer
###### Quick start
```bash
# create your environment file
echo CMS_API_URL='' >> .env
echo X_API_KEY='' >> .env
echo X_WRITE_API_KEY='' >> .env
...

# install dependencies
npm install

# start server
npm run dev

# build
npm run export
```
